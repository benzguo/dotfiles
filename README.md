## Installing:
* zsh (oh-my-zsh)
    * curl -L https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh | sh
* `./install`
* ag
    * `brew install the_silver_searcher`
* rupa/z
    * `cd /usr/bin`
    * `sudo curl -O https://raw.github.com/rupa/z/master/z.sh`
    * `sudo chmod 775 z.sh`
    * `. /usr/bin/z.sh`
